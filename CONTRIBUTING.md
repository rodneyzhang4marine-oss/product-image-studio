# Contributing

Thank you for considering a contribution to Product Image Studio.

## Development

```bash
npm install
npm run dev
```

Before opening a pull request:

```bash
npm run build
```

## Good First Contributions

- Improve documentation and screenshots.
- Add small ecommerce layout templates.
- Improve keyboard accessibility.
- Fix browser-specific UI issues.
- Add tests around canvas interactions.

## Privacy

Please do not commit private product photos, local absolute paths, customer data, credentials, or store-specific configuration.
